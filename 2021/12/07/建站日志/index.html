GitBash Upload

🔗git发布工程到github教程

实际步骤

  1. Git Bash Here 打开

  2. git init 初始化为git仓库

  3. git add . 添加文件

  4. git status 查看文件状态，绿色为已经准备好

    （此处出现not for commit，可能是文件夹下有内嵌的git仓库）

  5. git commit -m "备注" 提交文件到本地仓库

  6. git remote add origin https://github.com/VectorRo/VectorRo.github.io连接到远程仓库

  7. ssh -T git@github.com查看ssh状态

  8. git push -u origin master 复制到远程仓库（远程仓库为空）

    git push origin master复制到远程仓库（更新）
